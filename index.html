Cy
I'm Cy, Boris' personal chatbot! Say hello to me! (◠‿◠✿)

List of interactions:

  • Play Rock-Paper-Scissors!
    • Usage: Type a phrase similar to these ones to start playing!
      • Hey Cy, lets play Rock Paper Scissors!
      • Lets play RPS!
      • Hey Cy, lets play Rock Paper Scissors!
      To play again you can also type
      • Again!
      • One more!
      • Rematch!
  • Ask for a definition!
    • Usage: Type a phrase similar to these ones to receive information!
      • Hey Cy, what is Stack Overflow?
      • Hey Cy, what are heroes?
      • Hey Cy, what is a hyperbolic paraboloid?
      To receive more details about the questioned term type
      • Tell me more!
      • more info please!
      • more
  • Hear a joke!
    • Usage: Type a phrase similar to these ones to get some laughter!
      • Hey Cy, tell a joke
      • hey cy tell a funny joke!
      To receive additional jokes type
      • Another one!
      • more!
      • tell one more

Acknowledgements

The UI of this chatbot is based on this codepen, it was however heavily modified and extended. The chatbot-icon was taken from here.

The dataset used for training this chatbot is based on this as well as this project. However, the data was heavily reorganised, restructured and expanded on completely by hand. About 50% of the examples were cut and a large amount of own data points (questions, as well as answers) was added manually.
The list of jokes was taken from here.
